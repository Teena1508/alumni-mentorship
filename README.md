# 🎓 Alumni Mentorship Platform

A simple web-based platform that connects students with alumni mentors for guidance and career support.

## 🚀 Features
- Login page (basic authentication)
- Mentor listing with profiles
- Search mentors by name
- Individual profile page navigation
- Responsive card-based UI
- Interactive buttons (View Profile, Request Mentorship)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
